
# Telefram-bot is a Vue application.

Frontend Vue app for customer.


## Installation

Clone my project, after type in terminal:

```bash
  npm install 
  json-server db.json
  npm run serve
```

## Tech Stack

HTML, CSS, SCSS, Vue, Vue-router, Vuex, Axios, Json-server.
